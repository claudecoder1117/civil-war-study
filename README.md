# ⚔️ Sectionalism & the Civil War — Study Hub

A single-page study app for a U.S. History test covering **Sectionalism (Unit 10)** and the **Civil War (Unit 11)**.

**▶ Live site:** **https://claudecoder1117.github.io/civil-war-study/**

## Study modes
- 🃏 **Flashcards** — 33 key terms with definitions and why they mattered
- ✅ **Quiz** — multiple-choice with instant feedback (mirrors the test)
- 🔗 **Matching** — match terms to definitions
- 📜 **Timeline** — put the road-to-war events (1820–1865) in order
- ✍️ **Short Answer** — extended-response prompts with model answers
- 📖 **Study Sheet** — everything in one place, plus a cheat-facts box

## Run it
Just open `index.html` in any browser — it's fully self-contained and works **offline**. Your progress is saved locally in the browser (localStorage).

Optional local web server:

```bash
node server.js        # then open http://localhost:8899
```
